How to use this code?
First, open R studio and create a new project. Then, run 'All-results-....Rmd' for each forecasting approach, then run 'combined-results-and-plots.Rmd'.

Notes on data:
Provided norovirus data have random noise added due to data privacy. Raw norovirus data are available from UK Health Security Agency NoroOBK@ukhsa.gov.uk. 
The google trends data is not provided so all the pieces requiring Google Trends are commented out.
